# mypi

## Quick Clone

```bash
git clone --recurse-submodules https://github.com/xiaot623/mypi.git
```

## Submodules

- [pi-trace](pi-trace/): `https://github.com/xiaot623/pi-trace.git`

- [pi-subagents](pi-subagents/): `https://github.com/xiaot623/pi-subagents`

## Adding a Submodule

To add a new submodule to this repository, run the following command in the root directory:

```bash
git submodule add -b <branch-name> <repository-url> <local-path>
```
